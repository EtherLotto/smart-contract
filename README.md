# EtherLotto Smart Contract

## Start

```bash
clone https://github.com/EtherLotto/smart-contract
npm install

# development
touch .env & echo 'NODE_ENV = "development"' >> .env
touch .env.development && cat .env.example > .env.development

# production
touch .env & echo 'NODE_ENV = "production"' >> .env
touch .env.production && cat .env.example > .env.production
```
